# Day 12: Loop Challenges

This directory contains the Python loop challenges solved on Day 12.

![Loops Mastery Banner](loops_mastery_banner.png)

## 📂 Challenges List

*   **[`print_numbers_1_to_n.py`](print_numbers_1_to_n.py)**: Prints numbers from 1 to N.
*   **[`product_of_n_numbers.py`](product_of_n_numbers.py)**: Calculates the product of N input numbers.
*   **[`solid_rectangle.py`](solid_rectangle.py)**: Prints a solid rectangle of stars.
*   **[`sum_of_n_consecutive_numbers.py`](sum_of_n_consecutive_numbers.py)**: Calculates the sum of N consecutive integers starting from M.
*   **[`solid_rectangle_plus.py`](solid_rectangle_plus.py)**: Prints a solid rectangle of pluses.
*   **[`print_first_char_len_times.py`](print_first_char_len_times.py)**: Prints the first character of a word N times.
*   **[`left_aligned_star_triangle.py`](left_aligned_star_triangle.py)**: Prints a left-aligned star triangle.
*   **[`print_each_character.py`](print_each_character.py)**: Prints each character of a string on a new line.
*   **[`print_n_inputs.py`](print_n_inputs.py)**: Reads and prints N inputs sequentially.
*   **[`sum_of_n_inputs.py`](sum_of_n_inputs.py)**: Calculates the sum of N input numbers.
*   **[`print_numbers_m_to_n.py`](print_numbers_m_to_n.py)**: Prints numbers from M to N.
*   **[`solid_rectangle_no_spaces.py`](solid_rectangle_no_spaces.py)**: Prints a solid rectangle of stars without spaces.
*   **[`print_next_ten_numbers.py`](print_next_ten_numbers.py)**: Prints the next 10 numbers starting from N+1.
*   **[`sum_of_even_powers.py`](sum_of_even_powers.py)**: Calculates the sum of even powers of X up to 2N.
*   **[`sum_of_odd_powers.py`](sum_of_odd_powers.py)**: Calculates the sum of odd powers of X up to 2N-1.
*   **[`right_aligned_triangle_star_hash.py`](right_aligned_triangle_star_hash.py)**: Prints a right-aligned triangle with stars and a hash bottom row.
*   **[`hollow_rectangle.py`](hollow_rectangle.py)**: Prints a hollow rectangle of stars.
*   **[`star_zero_square.py`](star_zero_square.py)**: Prints a star-bordered square filled with zeros.
*   **[`number_square.py`](number_square.py)**: Prints a square of sequential numbers.
*   **[`star_zero_rectangle.py`](star_zero_rectangle.py)**: Prints a star-bordered rectangle filled with zeros.
*   **[`hollow_number_diamond.py`](hollow_number_diamond.py)**: Prints a hollow centered diamond pattern of numbers.
*   **[`butterfly_upper_half.py`](butterfly_upper_half.py)**: Prints the upper half of a hollow butterfly with a solid bottom row.
*   **[`right_aligned_star_triangle.py`](right_aligned_star_triangle.py)**: Prints a right-aligned triangle of stars.
*   **[`hollow_butterfly.py`](hollow_butterfly.py)**: Prints a hollow butterfly pattern of stars.
*   **[`hollow_inverted_pyramid.py`](hollow_inverted_pyramid.py)**: Prints a hollow inverted star pyramid.
